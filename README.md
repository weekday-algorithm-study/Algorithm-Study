## **스터디 규칙**

### **시간**

- 하루에 백준 Gold 1문제, Sliver 1문제

- 시간 복잡도
    - 단계마다 계산하고 설명
- 공간 복잡도
    - 사용한 자료구조마다 계산하고 설명
