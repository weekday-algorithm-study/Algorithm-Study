#풀이 방법
- y=-x 대칭 후 y축 대칭을 하면 시계방향으로 90도 회전한 결과와 같다.
  
#코드
'''C++
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        for (int i = 0; i<matrix.size(); i++){
            for (int j = 0; j<i; j++){
                swap(matrix[i][j],matrix[j][i]);
            }
        }
        for (int i = 0; i<matrix.size(); i++){
            reverse(matrix[i].begin(),matrix[i].end());
        }
    }
};
'''
