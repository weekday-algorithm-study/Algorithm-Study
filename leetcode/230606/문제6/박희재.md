## 접근 방식
dp 테이블을 사용하여 Bottom-up방식으로 채워나간다
공백을 두는 이유는 인덱스 1로 맞추기 위해서다
## 문제 풀이
```python
dp = [[0 for i in range(len(text2) + 1)] for j in range(len(text1) + 1)]
        
    text1 = " " + text1
    text2 = " " + text2
    
    for i in range(1, len(text1)):
        for j in range(1, len(text2)):
            if text1[i] == text2[j]:
                dp[i][j] = 1 + dp[i-1][j-1]
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
        
    return dp[-1][-1]
```
공백을 두는 이유는 인덱스 1로 맞추기 위해서다
